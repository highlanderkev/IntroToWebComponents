Examples
===

- Story Element to take input to create new Todo or defects
    - tap to create new todo/defect
    - click and drag/drop
- Todo Element that is editable, draggable, dropable, auto-updates
    - Tap to launch edit popup
    - Click and drag/drop
    - Auto updating state
- Grid Element that organizes story elements and todo elements
    - holds/contains many elements
    - allows for many categories (todo, in-progress, ready-for-test, done)
    - saves state configuration to local storage
